# Express Fundamentals API

A minimal Express.js API demonstrating RESTful principles, middleware, validation, and production-ready setup.

## Features

- ✅ RESTful CRUD operations on mock data
- ✅ Request validation using express-validator
- ✅ Custom middleware (logging, error handling)
- ✅ Environment configuration
- ✅ CORS, Helmet, Compression
- ✅ Nodemon for development
- ✅ Docker support (dev & prod)
- ✅ GitHub Actions CI/CD
- ✅ Thunder Client test collection
- ✅ PowerShell setup script

## Tech Stack

- Node.js 18+
- Express.js 4
- express-validator
- Docker
- GitHub Actions

## Getting Started

### Prerequisites

- Node.js 18+ or Docker
- npm or yarn

### Installation

```bash
# Clone repository
git clone <repo-url>
cd project-01-express-fundamentals

# Run setup script (PowerShell)
.\scripts\setup.ps1

# Or manually
npm install
cp .env.example .env
npm run dev
